Volume Rendering Fundamentals 2

Image Space is the 2D space in which the final image is displayed.

Object Space is the 3D space in which the data and sample volumes exist.

To render a volume, one can:

  • Either march through the whole object space, mapping each bit of color you come to to the image space with proper compositing (Forward Rendering).
  • Or, march through the image space, adding up all the object space color bits that contribute to a given image space point (Backward Rendering).
