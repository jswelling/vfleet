Pictures and Movies Generated with VFleet

Here is information on VFleet, a distributed volume rendering package for Unix workstations.

W.S. Lewellen, D.C. Lewellen and Aytekin Gel have run simulations of tornado vortices on the Pittsburgh Supercomputing Center's Cray C90. The following animation (MPEG format, 0.7 MB) shows one of their tornados.

This is an image of a head and brain generated from magnetic resonance data. The top of the skull has been removed in a preprocessing step. An animation of a slightly different view of this dataset is available in SGI movie format (1.7 MB) or in MPEG format (0.5 MB).

This is an image of an engine block generated from CT data. An animation is available in SGI movie format (1.6 MB) or in MPEG format (0.6 MB).

This image of the wave function for the highest occupied molecular orbital for Carbon 60 (Buckminsterfullerene) was generated from data produced by the Gaussian 92 ab initio quantum chemistry package. An animation of this dataset is available in SGI movie format (1.8 MB) or in MPEG format (0.5 MB).
